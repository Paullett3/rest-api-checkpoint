# 🚀 Node.js & Mongoose REST API Checkpoint

## ✅ Requirements Met
- **Env Config:** Variables loaded from `./config/.env`.
- **MVC Pattern:** User model exported from `models/User.js`.
- **CRUD Routes:** Full implementation of GET, POST, PUT, and DELETE.
- **Database:** Connected via MongoDB Atlas / Local.

## 🛠️ How to Test
1. Run `npm install`
2. Start server: `node server.js`
3. Use **Postman** to hit `localhost:3000/users`
